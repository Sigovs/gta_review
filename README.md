# GTA Motorcars — Design Review Prototype

An art-directed redesign concept of the GTA Motorcars USA home page — a private
collector-showroom identity (asymmetric hero, editorial catalogue, checkerboard
inventory, magazine lookbook, section-numbered signature system).

**This is a static design prototype for client review**, not the production build.
Tailwind and fonts load from a CDN, so an internet connection is required to view.

## View it

- **Locally:** open `index.html` in a browser.
- **Hosted (to send the client):** enable GitHub Pages on this repo
  (Settings → Pages → Deploy from branch → `main` / root). The live link will be:
  `https://sigovs.github.io/gta_review/`

## Notes

- Featured section: the right arrow cycles featured cars; the in-photo arrows
  page that car's gallery (gallery frames are stand-ins — swap for each car's
  real set).
- Recently Sold: a few extra cars are shown as archive stand-ins.
